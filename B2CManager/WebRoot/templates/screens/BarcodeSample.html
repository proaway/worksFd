JPEG Samples (Using BarcodeServlet)

You must have installed Barcode/Java before you can view this page. You may also need to change the path /examples/servlet in the BarcodeServlet URL reference so that it matches your configuration. Also this page must be served by a web server, i.e., you cannot open it directly (i.e. File|Open) from a web browser bypassing the web server (unless you change the img src attribute to point to a URL http://manager/BarcodeServlet in the HTML source of this page).

Here's the HTML content for the above images:

<img src="/manager/BarcodeServlet?image=3&type=6&data=123458">
<img src="/manager/BarcodeServlet?image=3&type=9&data=1234567&barwidth=2">
<img src="/manager/BarcodeServlet?image=3&type=3&data=12345678901&rotate=3">
<img src="/manager/BarcodeServlet?image=3&type=22&data=95020-342740&height=25">
